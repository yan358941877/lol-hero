## 项目介绍

这是一个使用 React 技术栈构建英雄联盟英雄背景、英雄皮肤、技能说明、使用技巧展示的单页应用

* 使用 Python 和 Beautiful Soup 爬取英雄联盟游戏中各英雄的相关数据
* 使用 React 来构建展示页面
* 使用 Redux 来进行状态管理
* 使用 redux-promise 中间件来处理异步流
* 使用 ReactCSSTransitionGroup、 classnames、 CSS 来实现动态效果
* 在开发过程中使用 ESLint 进行代码风格检测
* 使用 Express 来提供后端支持
